# Privacy Sentinel

An honest, in-browser privacy exposure audit — plus a downloadable Mac Deep Scan.
Every result is measured live on your device; anything that can't be measured says so.

**Live app:** https://techykinglj.github.io/s/

- Runs entirely in your browser (nothing stored, nothing sent beyond each check's described lookup)
- Installable to your phone's home screen (Safari: Share → Add to Home Screen)
- The Mac Deep Scan download is a read-only local tool: no installs, no network calls
